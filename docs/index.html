tunnelr. GitHub

Localhost to the world

Expose your local development server to the internet with your own domain. Self-hosted, unlimited tunnels, no restrictions.

Get Started View on GitHub 
# On your VPS - deploy the server
git clone https://github.com/Aakeeo/tunnelr.git && cd tunnelr
cp .env.example .env && nano .env
docker compose up -d

# On your laptop - connect
TUNNELR_SERVER=wss://yourdomain.com/ws tunnelr connect 3000

# That's it!
Tunnel established!
  Public URL: https://yourdomain.com/t/a1b2c3
  Forwarding: https://yourdomain.com/t/a1b2c3 -> http://localhost:3000

Why Tunnelr?

Self-hosted

Your server, your data. No third-party services involved.

Unlimited tunnels

Create as many tunnels as you need. No artificial limits.

Persistent URLs

Your tunnel URLs stay the same. No 2-hour expiry nonsense.

Custom domain

Use your own domain. Professional URLs for webhooks and demos.

Simple setup

One Docker command to deploy. One command to connect.

Automatic SSL

HTTPS works out of the box with Let's Encrypt.
